# 🌐 Interactive Navigation Menu

## 📌 Project Overview

The **Interactive Navigation Menu** is a responsive web application built using **HTML, CSS, and JavaScript**. The navigation bar remains fixed at the top of the page, changes its appearance when the user scrolls, and provides hover effects on menu items to improve the overall user experience.

This project demonstrates basic front-end web development concepts such as responsive layouts, DOM manipulation, event handling, and CSS transitions.

---

## ✨ Features

* 📌 Fixed navigation bar visible on all pages
* 🎨 Navigation bar changes style while scrolling
* 🖱️ Hover effects on navigation menu items
* 📱 Responsive design for different screen sizes
* ⚡ Smooth scrolling between sections
* 🎯 Clean and user-friendly interface

---

## 🚀 How to Run the Project

1. Download or clone this repository.

```bash
git clone https://github.com/your-username/your-repository-name.git
```

2. Open the project folder in **Visual Studio Code**.

3. Install the **Live Server** extension (recommended).

4. Right-click **index.html** and select **Open with Live Server**.

5. The application will open automatically in your default web browser.

---

## 📖 How to Use

1. Open the website in your browser.
2. Scroll down the page to see the navigation bar change its appearance.
3. Hover over the navigation links to view the interactive effects.
4. Click any navigation item to smoothly move to the corresponding section.
